# Contributing to zyphos-agents

Thank you for contributing. We review all PRs within 48 hours.

## How to contribute

1. Fork the repo
2. Create a branch: `git checkout -b feature/your-feature-name`
3. Make your changes
4. Run `npm run build` to verify everything compiles
5. Open a Pull Request with a clear description

## Good first contributions

- New agent blueprints in `examples/`
- Unit tests for any package
- README improvements or translations
- Bug fixes

## Code style

- TypeScript strict mode — all new code must compile with zero errors
- Add JSDoc comments on all public methods
- Follow the patterns established in existing packages
